## 摘  要

随着互联网科技的快速发展和海量数据的持续增长，搜索引擎系统已逐渐成为人们获取信息的主要途径。在面对庞大且繁杂的网络信息时，如何根据用户自身兴趣在某个特定领域内进行网页搜索且提升搜索效率，成为搜索引擎技术面临的重大挑战之一，这使得设计并实现一个内容定制化搜索引擎系统显得尤为重要。

关键词： 搜索引擎；C++；索引；内容定制化

## ABSTRACT

With the rapid development of internet technology and the continuous growth of massive data, search engine systems have gradually become the main means for people to     obtain information. When faced with vast and complex web information, how to conduct web searches within a specific field based on users' own interests and how to improve search efficiency have become major challenges for search engine technology. This makes the design and implementation of a content-customized search engine system particularly important.

To efficiently and accurately meet user needs, this project designs and implements a content-customized search engine system based on C++. The system is developed under the Linux operating system using the Reactor+Threadpool framework and is divided into offline preparation and online search parts. The offline preparation part involves using user-specified related topics' XML web pages as corpus, generating a deduplicated web page library from the corpus, and then creating a web page dictionary library and a web page index library to complete the preliminary work for the online search part. The online search part implements a keyword recommendation function through an improved edit matching algorithm and a multi-phrase query function to search for web information. Additionally, a caching system is designed to cache recent search results, ensuring the search engine's efficiency and persistence.

Keywords：Search Engine; C++; Index; Content customization
