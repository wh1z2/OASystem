# 表单设计器前端交互测试文档

## 1. 文档概述

### 1.1 测试目标
本文档为**阶段八表单设计器**提供完整的前端交互测试指引，覆盖 `FormDesigner.vue`（表单设计器页面）与 `DynamicForm.vue`（动态表单渲染组件）的所有用户交互场景、功能验证点、UI 元素检查及响应式行为验证。

### 1.2 测试范围
- **页面**: `FormDesigner.vue`（`/form-designer` 路由）
- **组件**: `DynamicForm.vue`、`ConfirmDialog.vue`
- **API 模块**: `formTemplateApi`（增删改查及分页）

### 1.3 测试环境
| 项目 | 要求 |
|------|------|
| 浏览器 | Chrome 120+ / Edge 120+ / Firefox 121+ |
| 分辨率 | 1920×1080（桌面端）、768×1024（平板端）、375×667（移动端） |
| 账号权限 | 具备 `form_design` 或 `all` 权限的角色 |

## 2. 页面结构与元素清单

### 2.1 列表模式（`mode === 'list'`）
| 元素 | 定位描述 | 类型 |
|------|---------|------|
| 页面标题 | "表单模板管理" | 文本 |
| 新建表单按钮 | 右上角蓝色按钮，含 "+" 图标 | 按钮 |
| 加载状态 | "加载中..." 文本 | 文本 |
| 空状态 | 文档图标 + "暂无表单模板" | 图文 |
| 模板列表项 | 名称、编码、描述、编辑/删除按钮 | 列表项 |
| 编辑按钮 | 每行右侧灰色 "编辑" 按钮 | 按钮 |
| 删除按钮 | 每行右侧红色 "删除" 按钮 | 按钮 |

### 2.2 设计模式（`mode === 'design'`）
| 区域 | 元素 | 类型 |
|------|------|------|
| 顶部栏 | 返回箭头、页面标题（新建/编辑）、预览按钮、保存按钮 | 按钮/文本 |
| 左侧面板 | 8 种表单组件（单行文本、多行文本、数字输入、邮箱输入、日期选择、下拉选择、单选框、多选框） | 可点击卡片 |
| 中间画布 | 字段列表、空状态提示、字段数量统计 | 容器/列表 |
| 字段卡片 | 字段名称输入框、字段标识输入框、选项文本域（条件）、占位提示输入框、必填复选框、上移/下移/删除按钮 | 复合组件 |
| 右侧面板 | 表单名称、表单编码、表单描述、状态下拉框 | 表单 |
| 预览弹窗 | 遮罩层、弹窗标题、关闭按钮、DynamicForm 渲染区、提交按钮 | 弹窗 |
| 确认弹窗 | 标题、消息、确认/取消按钮 | 弹窗 |
